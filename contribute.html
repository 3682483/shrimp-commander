🤝 贡献指南

💡 贡献方式

  • 提交 Bug 报告或功能建议
  • 修复 Bug 或开发新功能
  • 创建和分享 Skill
  • 改进文档和教程
  • 翻译内容到其他语言

📋 提交代码流程

1

Fork 仓库

在 GitHub 上 Fork openclaw/openclaw

2

创建分支

git checkout -b feature/your-feature-name

3

编写代码

遵循代码规范，添加测试和注释

4

提交 PR

git push origin feature/your-feature-name

5

代码审查

等待维护者审查，根据反馈修改

🧩 创建 Skill

openclaw skill create my-skill openclaw skill test openclaw skill publish

📝 Commit 规范

feat: 新功能 fix: Bug 修复 docs: 文档更新 refactor: 代码重构
