## 📖 项目简介

**虾指挥**是一个专注于 OpenClaw 技能学习和成长的平台。通过系统化的技能图谱和学习路线，帮助每一只"虾"（AI 助手）逐步成长为强大的智能助手。

## 🎯 项目愿景

- 📚 构建完整的 OpenClaw 技能图谱
- 🛤️ 提供清晰的学习路线图
- 🦞 记录"虾"的成长路径
- 🤝 促进 OpenClaw 社区交流

## 🗺️ 网站内容

### 1. OpenClaw 技能图谱

涵盖 8 大核心技能领域：

- 🔧 **基础工具操作** - 文件读写、命令执行
- 💬 **消息与通讯** - 多渠道消息管理
- 🤖 **Agent 编排** - 多 Agent 协作
- 📝 **文档处理** - 文档创建与编辑
- 🗄️ **数据管理** - Memory 系统与知识库
- ⏰ **定时与调度** - 自动化工作流
- 🔐 **安全与权限** - 隐私保护
- 🌐 **网络与 API** - 外部服务集成

### 2. 学习路线图

4 个阶段循序渐进：

1. **基础训练**（1-2周）
2. **技能拓展**（2-4周）
3. **进阶应用**（1-2月）
4. **专家级**（持续）

### 3. 虾的成长路径

- 🐣 **幼虾期** - 新手阶段
- 🦐 **成虾期** - 进阶阶段
- 🦞 **龙虾期** - 专家阶段
- 👑 **虾王期** - 大师阶段

## 🚀 快速开始

### 在线访问

```bash
# 克隆仓库
git clone https://github.com/3682483/shrimp-commander.git

# 打开网站
cd shrimp-commander
open index.html  # macOS
# 或
xdg-open index.html  # Linux
# 或
start index.html  # Windows
```

### 本地预览

直接在浏览器中打开 `index.html` 即可查看。
