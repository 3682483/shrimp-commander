常见问题

FREQUENTLY ASKED QUESTIONS

🔍

📦 安装与配置

OpenClaw 支持哪些操作系统？
支持 macOS 10.15+、Windows 10+、Linux (Ubuntu/Debian/CentOS)。需要 Node.js 18.0+。
如何安装 OpenClaw？
npm install -g openclaw 或使用 Docker：docker pull openclaw/openclaw
如何配置 API Key？
方式一：环境变量 export OPENAI_API_KEY=your-key
方式二：在项目根目录创建 .env 文件。⚠️ 切勿提交到 Git！
安装报错 permission denied 怎么办？
macOS/Linux：sudo npm install -g openclaw
Windows：以管理员身份运行 PowerShell

💬 使用问题

如何切换 AI 模型？
在 openclaw.config.js 中配置：
model: "claude-3-sonnet" / model: "gpt-4o" / model: "gemini-pro"
如何让 Agent 记住之前的对话？
OpenClaw 内置记忆系统。在 MEMORY.md 添加长期记忆，memory/ 目录存储每日笔记。系统自动语义检索。
如何安装第三方 Skill？
openclaw skill install skill-name 或从 GitHub：openclaw skill install github:user/repo
Agent 响应太慢怎么办？
1. 切换更快的模型
2. 检查网络/使用代理
3. 减少对话历史长度
4. 增加系统内存
如何限制 Agent 权限？
配置 Tool 白名单、文件系统隔离、网络限制、Skill 审核。在 openclaw.config.js 中设置安全策略。

🧩 技能开发

如何创建自己的 Skill？
1. openclaw skill create my-skill
2. 编辑 SKILL.md 添加描述和逻辑
3. openclaw skill test 测试
4. openclaw skill publish 发布
SKILL.md 应该包含什么？
name、description、trigger（触发条件）、tools（工具列表）、instructions（执行指令）、examples（示例）
如何调试 Skill？
openclaw start --debug 日志模式
对话中使用 /step 单步执行
Web Dashboard 可视化调试

🚀 部署相关

如何部署到服务器？
Docker 部署：
docker run -d -p 3000:3000 -v $(pwd)/data:/app/data openclaw/openclaw
如何配置 HTTPS？
推荐使用 Nginx + Let's Encrypt 或 Caddy（自动证书）。Docker 可用 traefik。
如何实现多用户隔离？
每个 Agent 实例隔离运行，独立记忆和配置。企业版提供完善多租户管理。

💰 费用相关

OpenClaw 本身收费吗？
开源版完全免费！ MIT 许可，可商用。企业版提供额外功能和技术支持。
使用需要付 AI 模型费吗？
是的，需自付 API 费用。Claude/GPT 按 token 计费，Gemini 有免费额度，开源模型可本地免费部署。
如何查看和控制 API 费用？
Web Dashboard 提供实时用量统计、费用估算、预算限制、超支告警。
