⚡ 5 分钟快速开始

# 安装 OpenClaw CLI npm install -g openclaw # 创建新项目 openclaw init my-agent # 进入项目目录 cd my-agent # 启动 Agent openclaw start

🎉 就这么简单！你的第一个 AI Agent 已经运行起来了。访问 http://localhost:3000 查看 Web 界面。
